Autolink
========

Yes
-----

~~~html
<p>http://example.com</p>
~~~

~~~html
<li>http://example.com</li>
~~~

No
-----

~~~html
<code>http://example.com</code>
~~~

Result
------

~~~html
<li><a href="http://example.com">http://example.com</a></li>
~~~
